# AHMEDMART Backend

This is the backend server for the AHMEDMART application built with Node.js, Express, and MongoDB.

## Features

- User registration, login, and token refresh
- Product, Vendor, Zone, Review, Order, Notification management
- WhatsApp integration for messages, order updates, and OTP
- JWT authentication and password hashing
- MongoDB database using Mongoose
